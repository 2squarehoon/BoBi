### BOBI_BACKEND

- Django ver. 3.2.12 사용

```bash
$ pip install django==3.2.12
```

- DB: mysql 사용

- 시작 전 가상환경

  - 생성

  ```bash
  $ python -m venv venv
  ```

  - 실행

  ```bash
  $ source venv/Scripts/activate
  ```

- requirements.txt

  - 설치 후 추가

  ```bash
  $ pip freeze > requirements.txt
  ```

  - 전체 설치

  ```bash
  $ pip install -r requirements.txt
  ```
