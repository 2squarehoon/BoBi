# MQTT subscribe
- subscribe MQTT message from web
    - move
    - S3 message sent flag

## How to run
1. activate virtual env
    ```
    $ cd ~/google_stt
    $ source env/bin/activate
   ```
2. run the code
    ```
    $ (env) cd ~/WAVEGO/RPi
    $ (env) python ./mqtt_subscribe.py --user_id [user id]
    ```
